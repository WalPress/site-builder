# 🌐 Walpress

**Walpress** is a decentralized website builder for [Walrus](https://walrus.network), designed to let anyone create, manage, and deploy static or decentralized sites — all from their own machine. No servers, no signups, no hosting fees. Just full control of your content, domain, and identity.

---

## 🚀 What is Walpress?

Walpress is like WordPress… but for the decentralized web. It allows users to:

- Build static or decentralized websites using a drag-and-drop editor.
- Deploy those websites directly to the **Walrus decentralized storage network**.
- Connect a custom **SuiNS domain** to your site.
- Manage everything **offline or locally** — your data never leaves your machine unless you publish it.

---

## 🔑 Features

- 🔐 **Local Wallet Generation**  
  Securely generate a Sui wallet directly in-app. Your private key stays local and encrypted.

- 💾 **Downloadable Private Key**  
  Download your private key as a file. A persistent reminder ensures you backup safely.

- 🔄 **Persistent Sessions**  
  Automatically re-login with stored wallet or import private key file.

- 🧱 **Multiple Login Options**  
  - Connect with Slush  
  - Connect with Browser Wallet  
  - Connect Locally (Generate or Select Wallet Profile)

- 🏗️ **Build Static or Decentralized Websites**  
  Use the WYSIWYG builder to visually create a website. Design, edit, and preview everything locally.

- 🚀 **One-Click Deployment to Walrus**  
  Publish your site directly to Walrus in a click. Receive a Blob ID once deployed.

- 🌐 **SuiNS Domain Linking**  
  Attach a SuiNS domain to any of your deployed sites. Easily browse and manage all your owned domains.

- 📁 **Site Dashboard Management**  
  View, edit, delete, and deploy your Walpress sites. See status (deployed / undeployed), Blob ID, and actions at a glance.

- 🛠️ **Advanced Settings**  
  Change RPC URLs and configure gas budget or other developer options.

- 🧩 **Fully Offline-Capable**  
  All actions — from site building to wallet generation — are local-first. Ideal for privacy-focused or low-internet environments.
